## ðŸš€ Installation Guide

### <img src="https://raw.githubusercontent.com/alrra/browser-logos/master/src/chrome/chrome_48x48.png" width="24" /> Google Chrome / <img src="https://raw.githubusercontent.com/alrra/browser-logos/master/src/brave/brave_48x48.png" width="24" /> Brave / <img src="https://raw.githubusercontent.com/alrra/browser-logos/master/src/edge/edge_48x48.png" width="24" /> Microsoft Edge

1.  Open your browser and navigate to the extensions page:
    - **Chrome/Brave:** `chrome://extensions`
    - **Edge:** `edge://extensions`
2.  Enable **Developer mode** (usually a toggle switch in the top-right or bottom-left corner).
3.  Click the **Load unpacked** button.
4.  Select the **folder** you extracted in the "Download" step.
5.  The extension should now be active!

### <img src="https://raw.githubusercontent.com/alrra/browser-logos/master/src/firefox/firefox_48x48.png" width="24" /> Mozilla Firefox

1.  Open Firefox and navigate to: `about:debugging#/runtime/this-firefox`
2.  Click the **Load Temporary Add-on...** button.
3.  Navigate to the extracted folder and select the `manifest.json` file.
4.  The extension is now running.
    - _Note: As a temporary add-on, it will be removed if you restart Firefox._

### <img src="https://raw.githubusercontent.com/alrra/browser-logos/master/src/opera/opera_48x48.png" width="24" /> Opera

1.  Open usage `opera://extensions`.
2.  Enable **Developer mode** (top-right corner).
3.  Click **Load unpacked**.
4.  Select the extracted folder.
